Message app powered by React, Redux and Firebase
===================


Just an app to test these technologies.
Message app.

**Tech used**
 - ReactJs 
 - Redux
 - Blendid (Task and builds tools)

----------


Command
-------------

    yarn

Install project.


    yarn run blendid


Launch dev server


    yarn run blendid -- build

Build app for production (in public folder)
